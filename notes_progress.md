### 一、数据中心
- [x] 地址输入和地图定位
- [x] 餐厅项目选择
- [x] 商场/商铺名称输入

### 二、现场勘察
- [x] 铺位情况
  - 可视性选择
  - 是否被遮挡
  - 通达性（楼层、门头宽度、扶梯旁、楼层中间、主通道、热门品牌旁）
  - 缺陷情况（房型缺陷、工程条件缺陷、需上台阶、进深巷）
- [x] 客流量市调
  - 3个品牌（A/B/C）
  - 每个品牌：名称、客单价区间
  - 周一/周五/周六的午市和晚市上座率
- [x] 商场品牌业绩参考
  - 3个品牌的业绩情况
  - 品牌名称、品类、月营业额、日翻台率、备注
- [x] 选址评分（手动填写）
  - 客流量 1-10分
  - 铺位条件 1-10分
  - 客群匹配 1-10分
  - 成本控制 1-10分
  - 竞争环境 1-10分
